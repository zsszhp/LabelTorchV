# 参考项目

本目录存放供技术路线和实现参考的开源项目。按深度1克隆，仅保留最新提交。

## 项目清单

| 项目 | 用途 | 优先级 |
|------|------|--------|
| X-AnyLabeling | YOLO标注集成、标注工具交互组织 | P1 |
| Labelme | 轻量本地标注工作流、快捷键与基础交互 | P1 |
| Ultralytics | 训练API设计、模型导出流程 | P1 |
| CVAT | 复杂标注流程组织方式、标注审核思路 | P2 |
| roLabelImg | OBB与旋转框表达、旋转框标注交互 | P2 |

## 使用说明

- 参考技术路线和项目管理方式，不盲目复制功能
- 标注交互细节参考 Labelme、X-AnyLabeling
- 训练编排参考 Ultralytics 官方实现
