# V1.0 — 标注引擎版（BBox）· 任务书

> ⭐ 重大里程碑：完成后软件具备完整的标注能力。

## 1. 版本目标

高性能图像画布 + BBox 标注编辑 + 标签读写 + 快捷键体系。用户能在软件中直接创建、编辑、删除检测框并保存为 YOLO txt 格式。

## 2. 依赖

V0.6 全部验收通过。

## 3. 边界定义

**本版本包含**：
- HBB（水平矩形框）标注的完整交互链路
- 图像缩放、平移、高分辨率图片流畅浏览
- 标签原子写回（先写临时文件，再替换）
- 样本快速切换（前后翻页）

**本版本不包含**：
- OBB（旋转矩形框）→ V3.0
- 多边形标注 → 后续版本
- 辅助标注候选框 → V1.1
- 撤销/重做 → 后续增强
- 批量标注操作 → 后续增强

### 任务 1.0.1 — 标注画布核心（C++ QQuickItem）

**交付物**：`src/features/annotation/AnnotationCanvas.h/.cpp`

这是标注引擎的核心渲染组件，使用 C++ 实现以获得最佳性能。

```cpp
class AnnotationCanvas : public QQuickPaintedItem {
    Q_OBJECT
    Q_PROPERTY(QString imagePath READ imagePath WRITE setImagePath NOTIFY imageChanged)
    Q_PROPERTY(qreal zoomLevel READ zoomLevel WRITE setZoomLevel NOTIFY zoomChanged)
    Q_PROPERTY(QPointF panOffset READ panOffset WRITE setPanOffset NOTIFY panChanged)
    Q_PROPERTY(int boxCount READ boxCount NOTIFY boxesChanged)
    Q_PROPERTY(bool isDirty READ isDirty NOTIFY dirtyChanged)
    Q_PROPERTY(QString currentTool READ currentTool WRITE setCurrentTool NOTIFY toolChanged)

public:
    explicit AnnotationCanvas(QQuickItem *parent = nullptr);

    // 图像加载
    void setImagePath(const QString &path);

    // 缩放控制（0.1 ~ 10.0）
    void setZoomLevel(qreal level);
    Q_INVOKABLE void zoomIn();      // +10%
    Q_INVOKABLE void zoomOut();     // -10%
    Q_INVOKABLE void zoomFit();     // 适应窗口
    Q_INVOKABLE void zoomActual();  // 100%

    // 框操作
    Q_INVOKABLE void addBox(int classId, qreal x, qreal y, qreal w, qreal h);
    Q_INVOKABLE void deleteSelectedBox();
    Q_INVOKABLE void setSelectedBoxClass(int classId);
    Q_INVOKABLE int selectedBoxIndex() const;

    // 标签加载/保存
    Q_INVOKABLE bool loadLabels(const QString &labelPath);
    Q_INVOKABLE bool saveLabels(const QString &labelPath);

    // 工具切换
    // "select" — 选择/移动/缩放框
    // "create" — 创建新框
    // "pan"    — 拖拽平移画布
    void setCurrentTool(const QString &tool);

signals:
    void imageChanged();
    void zoomChanged();
    void panChanged();
    void boxesChanged();
    void dirtyChanged();
    void toolChanged();
    void boxSelected(int index, int classId, qreal x, qreal y, qreal w, qreal h);
    void boxDeselected();

protected:
    void paint(QPainter *painter) override;
    void mousePressEvent(QMouseEvent *event) override;
    void mouseMoveEvent(QMouseEvent *event) override;
    void mouseReleaseEvent(QMouseEvent *event) override;
    void wheelEvent(QWheelEvent *event) override;

private:
    QImage m_image;
    QList<BoundingBox> m_boxes;     // 当前图片的所有框
    int m_selectedIndex = -1;
    QString m_currentTool = "select";
    qreal m_zoom = 1.0;
    QPointF m_pan = {0, 0};
    bool m_isDirty = false;

    // 坐标转换：图像坐标 ↔ 视口坐标
    QPointF imageToViewport(const QPointF &imgPt) const;
    QPointF viewportToImage(const QPointF &vpPt) const;

    // 命中测试：判断点击位置是否在某个框的边角上
    enum HitType { None, Inside, TopLeft, TopRight, BottomLeft, BottomRight,
                   Left, Right, Top, Bottom };
    HitType hitTest(const QPointF &imgPt, int boxIndex) const;
};
```

**BoundingBox 数据结构**：

```cpp
struct BoundingBox {
    int classId;
    qreal cx, cy, w, h;      // YOLO 归一化坐标 (0~1)
    QString className;
    QColor color;
    bool isConfirmed = true;  // 辅助标注用，V1.1 启用
    float score = 1.0f;       // 辅助标注置信度，V1.1 启用
    QString sourceType = "manual"; // manual / assisted_confirm / assisted_edited
};
```

**渲染规则**：
1. 背景层：原始图像（根据 zoom/pan 变换）
2. 标注层：所有 BBox（classId 对应颜色，选中框加粗+控制点）
3. 交互层：创建中的临时框（虚线）、选中框的 8 个控制点
4. 标签层：每个框左上角显示"类别名 置信度"文本

**性能要求**：
- 4000×3000 图片缩放平移帧时间 ≤ 16ms（60fps）
- 100 个框同时渲染无明显卡顿
- 使用 QImage 缓存已加载图片，避免重复解码

### 任务 1.0.2 — 标签文件读写

**交付物**：`src/features/annotation/LabelIO.h/.cpp`

```cpp
class LabelIO {
public:
    // 读取 YOLO txt 标签
    static QList<BoundingBox> loadYoloTxt(const QString &path,
                                          const QMap<int, QString> &classNames,
                                          const QMap<int, QColor> &classColors);

    // 保存 YOLO txt 标签（原子写回）
    // 1. 写入 path + ".tmp"
    // 2. 校验 .tmp 可再次解析
    // 3. 原子替换正式文件
    // 4. 失败时保留旧文件
    static bool saveYoloTxt(const QString &path, const QList<BoundingBox> &boxes,
                             int imageWidth, int imageHeight);

    // 坐标校验：所有值在 [0, 1] 范围内
    static bool validateBox(const BoundingBox &box);
};
```

**原子写回流程**：
```
1. 生成临时文件 label.txt.tmp
2. 写入所有框数据
3. 重新读取 .tmp 文件验证可解析
4. 验证通过 → 删除旧 label.txt → 重命名 .tmp → label.txt
5. 验证失败 → 删除 .tmp → 报错 → 旧文件不受影响
```

class SampleNavigator : public QObject {

### 任务 1.0.5 — 快捷键系统

| 快捷键 | 功能 | 上下文 |
|---|---|---|
| `W` | 切换到画框工具 | 标注页 |
| `V` | 切换到选择工具 | 标注页 |

| `Ctrl+0` | 缩放适应窗口 | 标注页 |
| `Ctrl+1` | 缩放到 100% | 标注页 |
| `Escape` | 取消当前操作/取消选中 | 标注页 |

## 6. 验收检查单

1. 打开 4000×3000 图片，缩放平移流畅（无明显卡顿）
2. 画框 → 保存 → 切换到其他图片 → 切回 → 框位置一致
3. 创建 10 个框 → 全部保存 → 用文本编辑器打开 .txt 验证格式正确
4. 拖拽移动框 → 保存 → 坐标正确更新
5. 拖拽边角缩放框 → 保存 → 尺寸正确更新
6. 快捷键 1~9 切换类别 → 保存 → class_id 正确
7. 连续 A/D 键切换 100 张样本 → 无崩溃无卡顿
8. 修改未保存时切换样本 → 弹出保存提示
9. 原子写回：保存过程中断电/崩溃不丢失旧标签
10. Delete 键删除框 → 保存 → 该框不在标签文件中
