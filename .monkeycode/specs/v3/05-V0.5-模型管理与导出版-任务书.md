### 任务 0.5.2 — 模型版本服务

```cpp
struct ModelVersionInfo {
    QString id;
    QString runId;
    QString parentModelVersionId;
    QString modelFamily;
    QString modelVariant;
    QString bestWeightPath;
    QJsonObject metrics;   // mAP50, mAP50-95, precision, recall
    QStringList tags;      // baseline, best-so-far, production-candidate
    QDateTime createdAt;

class ModelVersionService : public QObject {
    Q_OBJECT
public:
    Q_INVOKABLE QList<ModelVersionInfo> listVersions(const QString &projectId);
    Q_INVOKABLE ModelVersionInfo getVersion(const QString &versionId);
    Q_INVOKABLE bool addTag(const QString &versionId, const QString &tag);
    Q_INVOKABLE bool removeTag(const QString &versionId, const QString &tag);
    Q_INVOKABLE QList<ModelVersionInfo> getLineage(const QString &versionId);
};
```

### 任务 0.5.3 — Python 导出 Handler

**IPC 命令 `export.run`**：

请求：
```json
{
    "command": "export.run",
    "payload": {
        "model_path": "/path/to/best.pt",
        "format": "onnx",
        "output_dir": "/path/to/exports",
        "options": {
            "opset": 13,
            "dynamic": true,
            "simplify": true,
            "imgsz": [640, 640]
        }
    }
}
```

响应：
```json
{
    "success": true,
    "result": {
        "output_path": "/path/to/exports/best.onnx",
        "file_size_bytes": 12345678,
        "validation": {
            "loadable": true,
            "input_shape": [1, 3, 640, 640],
            "output_count": 1
        }
    }
}
```

**导出后验证逻辑**：
- pt 导出：检查文件存在 + torch.load 可加载
- onnx 导出：检查文件存在 + onnxruntime.InferenceSession 可加载 + 输入输出形状正确
