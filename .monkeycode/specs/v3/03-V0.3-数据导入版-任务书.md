### 任务 0.3.2 — 数据导入服务

**交付物**：`src/features/dataset/DatasetService.h/.cpp`

```cpp
struct ImportResult {
    int totalImages;
    int matchedPairs;       // 图片+标签配对成功
    int missingLabels;      // 有图片无标签
    int emptyLabels;        // 标签文件为空
    int invalidLabels;      // 标签格式错误
    QStringList errorFiles; // 异常文件列表
};

class DatasetService : public QObject {
    Q_OBJECT
public:
    // 扫描目录并导入
    // imageDir: 图片目录
    // labelDir: 标签目录（可与图片目录相同或为子目录）
    // datasetName: 数据集名称
    Q_INVOKABLE void startImport(const QString &imageDir, const QString &labelDir,
                                  const QString &datasetName);

    // 获取数据集的样本列表模型
    Q_INVOKABLE QAbstractListModel* sampleModel(const QString &datasetId);

    // 获取类别列表
    Q_INVOKABLE QJsonArray getClasses(const QString &datasetId);

    // 更新类别名称
    Q_INVOKABLE bool updateClassName(const QString &taxonomyId, int classId, const QString &name);

signals:
    void importProgress(int current, int total, const QString &currentFile);
    void importFinished(const ImportResult &result);
    void importError(const QString &message);
};
```

**YOLO txt 解析规则**：
- 图片格式：.jpg, .jpeg, .png, .bmp, .tif, .tiff
- 标签文件：与图片同名 + .txt
- 每行格式：`<class_id> <x_center> <y_center> <width> <height>`
- 坐标范围校验：0.0 ~ 1.0
- class_id 校验：非负整数
- 空行跳过，格式错误记录到 error_code

**导入流程**：
1. 扫描 imageDir 收集图片文件列表
2. 对每张图片在 labelDir 查找同名 .txt
3. 解析标签文件，校验格式
4. 提取所有出现的 class_id，生成 taxonomy
5. 读取图片尺寸（宽高）
6. 写入 dataset_samples 表
7. 生成缩略图到 cache/thumbnails/
