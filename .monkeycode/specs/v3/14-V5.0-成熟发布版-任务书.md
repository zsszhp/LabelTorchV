### 任务 5.0.1 — 打包流水线脚本

**交付物**：`scripts/build_release.py`

完整打包流程：

```python
"""
标炬发布包构建脚本

用法:
    python scripts/build_release.py --variant cpu
    python scripts/build_release.py --variant cuda
    python scripts/build_release.py --variant all

输出:
    dist/
    ├── LabelTorch-v5.0.0-win64-cpu-setup.exe     # Qt IFW 安装包
    ├── LabelTorch-v5.0.0-win64-cuda-setup.exe
    ├── LabelTorch-v5.0.0-win64-cpu-portable.7z    # 绿色解压包
    └── LabelTorch-v5.0.0-win64-cuda-portable.7z
"""
```

**构建步骤**：

1. **CMake Release 构建**
   ```
   cmake --preset msvc2022-release
   cmake --build --preset msvc2022-release --config Release
   ```

2. **windeployqt 收集 Qt 依赖**
   ```
   windeployqt --qmldir src/ --release LabelTorch.exe
   ```

3. **打包 Python 嵌入式环境**
   ```
   # 方案 A：conda-pack
   conda activate labeltorch
   conda pack -n labeltorch -o labeltorch_env.tar.gz

   # 方案 B：embed Python
   # 下载 Python 3.11 embedded distribution
   # + 安装 pip + 安装依赖到 Lib/site-packages
   ```

4. **整合目录结构**
   ```
   release/
   ├── LabelTorch.exe
   ├── Qt DLLs + QML 模块 + 插件
   ├── python/
   │   ├── python.exe
   │   ├── Lib/site-packages/  (torch, ultralytics, onnxruntime, ...)
   │   └── ...
   ├── resources/
   │   └── icons/
   └── check_env.bat          # 环境自检脚本
   ```

5. **Qt IFW 打包**
   ```
   # 创建 Qt IFW 配置
   config/
   ├── config.xml             # 安装器全局配置
   └── packages/
       ├── com.labeltorch.core/
       │   └── meta/installscript.qs
       ├── com.labeltorch.python/
       └── com.labeltorch.cuda/     # 可选组件
   ```

6. **7z 绿色包**
   ```
   7z a -mx9 LabelTorch-v5.0.0-win64-cpu-portable.7z release/
   ```

### 任务 5.0.2 — 环境自检脚本

**交付物**：`scripts/check_env.bat` + `scripts/check_env.py`

```batch
@echo off
echo ========================================
echo  标炬 LabelTorch 环境自检
echo ========================================
echo.
echo [1/5] 检查 NVIDIA 驱动...
nvidia-smi >nul 2>&1
if %errorlevel% neq 0 (
    echo [警告] 未检测到 NVIDIA 驱动，将以 CPU 模式运行
) else (
    echo [通过] NVIDIA 驱动已安装
    nvidia-smi --query-gpu=name,driver_version --format=csv,noheader
)
echo.
echo [2/5] 检查 Python 环境...
python\python.exe scripts\check_env.py
echo.
echo [3/5] 检查 PyTorch...
python\python.exe -c "import torch; print(f'PyTorch {torch.__version__}, CUDA: {torch.cuda.is_available()}')"
echo.
echo [4/5] 检查 Ultralytics...
python\python.exe -c "from ultralytics import YOLO; print('Ultralytics OK')"
echo.
echo [5/5] 检查 ONNX Runtime...
python\python.exe -c "import onnxruntime; print(f'ORT {onnxruntime.__version__}')"
echo.
echo ========================================
echo  自检完成
echo ========================================
pause
```
