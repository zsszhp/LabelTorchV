# V3.0 — OBB 标注版 · 任务书

## 1. 版本目标

支持旋转矩形框（OBB）标注 + OBB 数据格式读写 + YOLOv8-OBB 训练。

## 2. 依赖

V2.0 全部验收通过。

## 3. 边界定义

**本版本包含**：
- RotatedBox 几何图元
- OBB 创建交互（画框 + 旋转手柄）
- OBB 编辑（旋转、移动、缩放）
- DOTA 格式标签读写
- YOLOv8-OBB 适配器
- OBB 训练与评估

**本版本不包含**：
- 多边形标注 → 后续
- OBB 辅助标注（推理产生旋转框） → 可复用 V1.1 框架

## 4. 任务分解

### 任务 3.0.1 — RotatedBox 数据结构

**交付物**：扩展 `BoundingBox` 或新增 `RotatedBoundingBox`

```cpp
struct RotatedBoundingBox {
    int classId;
    // 四点坐标（DOTA 格式原始表示）
    QPointF p1, p2, p3, p4;    // 四个角点（归一化坐标）
    // 等价中心表示

    void fromPoints(QPointF p1, QPointF p2, QPointF p3, QPointF p4);
    void toPoints();

    // 最小外接矩形
    QRectF boundingRect() const;
};
```

### 任务 3.0.3 — OBB 画布交互

扩展 `AnnotationCanvas`，增加 OBB 模式：

**OBB 创建交互**：
1. 用户按住鼠标拖拽 → 画出初始水平矩形
2. 松开鼠标 → 在框的一侧出现旋转手柄
3. 拖拽旋转手柄 → 框跟随旋转

**OBB 编辑交互**：
- 4 个角点可独立拖拽（改变形状）
- 中心拖拽移动整个框
- 旋转手柄：框一条边中点外侧的小圆，拖拽旋转

**渲染**：
- OBB 框使用四边形绘制（QPainter::drawPolygon）
- 选中时 4 个角点显示控制点
- 旋转手柄显示为边外的小圆 + 连线

```cpp
// AnnotationCanvas 新增
```

## 6. 验收检查单

1. OBB 创建：画框 + 旋转手柄旋转 → 四角坐标正确
2. OBB 保存：YOLO-OBB 格式写入标签文件 → 8 个坐标值正确归一化
3. OBB 重载：保存后切换图片再切回 → 框位置角度一致
4. OBB 编辑：拖拽角点改变形状 → 保存 → 坐标正确更新
5. DOTA 格式导入/导出正确
6. YOLOv8-OBB 训练 1 epoch → 成功完成
7. OBB 训练结果 mAP(OBB) 指标正确显示
8. HBB 和 OBB 模式切换不影响已有标注
