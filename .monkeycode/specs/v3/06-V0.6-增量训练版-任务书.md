# V0.6 — 增量训练版 · 任务书

## 1. 版本目标

支持选择历史模型权重进行增量训练，追加导入新图片，模型谱系链可视化。

## 2. 依赖

V0.5 全部验收通过。

## 3. 任务分解

### 任务 0.6.1 — 训练参数面板扩展

在训练工作台页面增加"预训练权重"选择器：

**三种训练模式**：

| 模式 | UI 展示 | pretrained_weights 值 |
|---|---|---|
| 从零训练 | 单选项"不使用预训练权重" | `null` |
| 预训练权重 | 单选项"使用官方预训练权重" | `"yolov8n.pt"` 等 |
| 增量训练 | 单选项"基于历史版本继续训练" + 版本选择下拉框 | 历史 best.pt 路径 |

**版本选择下拉框**：
- 列出当前项目所有 ModelVersion
- 每项显示：版本号 + 模型变体 + mAP50-95
- 选择后自动填充 pretrained_weights 路径

---

### 任务 0.6.2 — 增量训练模型注册

训练完成后模型注册逻辑扩展：
- 如果是增量训练，`model_versions.parent_model_version_id` 填写父版本 ID
- 训练配置快照中记录 `training_mode: "incremental"` 和 `parent_version_id`

---

### 任务 0.6.3 — 模型谱系链视图

**交付物**：`src/features/model/LineageView.qml`

在模型版本页面新增"谱系"标签页，展示模型的父子关系：

```
V1 (yolov8n, mAP=0.75)
 └── V2 (yolov8n, mAP=0.82)
      └── V3 (yolov8s, mAP=0.89)
           └── V4 (yolov8s, mAP=0.91)  ← 当前最优
```

使用 TreeView 或自定义 QML 绘制。每个节点显示版本号、模型变体、关键指标。

---

### 任务 0.6.4 — 追加导入

在数据导入页面增加"追加导入"功能：
- 对已有数据集追加新图片
- 扫描新目录，自动跳过已存在的图片（按文件名或 hash 去重）
- 更新 dataset_samples 表和 sample_count
- 追加导入后提示用户可重新训练

---

### 任务 0.6.5 — 重新划分 train/val

在训练工作台增加"重新划分"按钮：
- 弹出对话框：输入划分比例（默认 0.8）
- 使用新的随机种子重新划分
- 保留原有配对关系，只改变 train/val 分配

---

## 4. 验收检查单

1. 选择"增量训练" → 选择 V1 权重 → 训练启动 → 完成
2. 新模型版本的 parent_model_version_id = V1 的 ID
3. 谱系链视图正确展示父子关系
4. 追加导入 20 张新图片 → 样本数增加 → 去重正确
5. 重新划分 train/val → 训练使用新划分 → 训练正常
6. 增量训练后 mAP 指标有所提升（功能验证，不要求必须提升）
