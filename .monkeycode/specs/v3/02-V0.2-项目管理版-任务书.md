# V0.2 — 项目管理版 · 任务书

## 1. 版本目标

能创建、打开、切换项目。每个项目是一个独立目录，有自己的 SQLite 数据库和文件结构。

## 2. 依赖

V0.1 全部验收通过。

## 3. 任务分解

### 任务 0.2.2 — 项目服务

**交付物**：`src/features/project/ProjectService.h/.cpp`

```cpp
struct ProjectInfo {
    QString projectId;    // UUID
    QString name;
    QString rootPath;
    QString defaultDevice;      // "cuda:0" / "cpu"
    QString defaultModelFamily; // "yolov8"
    QDateTime createdAt;
    QDateTime updatedAt;
};

class ProjectService : public QObject {
    Q_OBJECT
    Q_PROPERTY(ProjectInfo current READ currentProject NOTIFY currentProjectChanged)
    Q_PROPERTY(bool hasOpenProject READ hasOpenProject NOTIFY currentProjectChanged)

public:
    // 创建新项目
    // 1. 创建目录结构  2. 初始化数据库  3. 写入 project.json
    Q_INVOKABLE bool createProject(const QString &name, const QString &parentDir,
                                    const QString &device, const QString &modelFamily);

    // 打开已有项目（从 project.json 读取）
    Q_INVOKABLE bool openProject(const QString &projectRootPath);

    // 关闭当前项目
    Q_INVOKABLE void closeProject();

    ProjectInfo currentProject() const;
    bool hasOpenProject() const;

signals:
    void currentProjectChanged();
    void projectError(const QString &message);
};
```

**项目目录结构创建**：

```
<project_root>/
├── project.json          # 项目元信息
├── labeltorch.db         # SQLite 数据库
├── data/
│   ├── datasets/         # 导入数据集引用
│   ├── snapshots/        # 数据快照
│   ├── taxonomy/         # 类别体系
│   └── revisions/        # 标签修订
├── models/
│   ├── runs/             # 训练运行输出
│   └── versions/         # 注册模型版本
├── exports/              # 导出产物
├── cache/
│   ├── thumbnails/       # 缩略图缓存
│   └── temp/             # 临时文件
└── logs/                 # 日志
```

### 任务 0.2.5 — 新建项目对话框

**交付物**：`src/features/project/CreateProjectDialog.qml`

**表单字段**：

| 字段 | 类型 | 默认值 | 校验 |
|---|---|---|---|
| 项目名称 | 文本输入 | — | 非空，不含特殊字符 |
| 项目路径 | 文件夹选择 | F:/projects | 路径可写 |
| 默认设备 | 下拉框 | cuda:0 | 自动检测可用设备 |
| 默认模型 | 下拉框 | yolov8 | 枚举值 |

**行为**：
- 点击"创建" → 调用 ProjectService.createProject()
- 成功后自动打开项目，跳转到项目概览页
- 失败显示错误提示

## 4. 验收检查单

1. 点击"新建项目" → 填写信息 → 创建成功
2. 项目目录结构正确（所有子目录存在）
3. project.json 内容正确
4. labeltorch.db 存在且表结构正确
5. 关闭程序 → 重新打开 → 最近项目列表显示刚创建的项目
6. 点击最近项目 → 项目打开成功，状态栏更新
7. 修改项目设置 → 保存 → 重新打开验证
