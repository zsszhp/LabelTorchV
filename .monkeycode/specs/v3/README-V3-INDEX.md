## AI 实现者须知

1. **严格按版本序实现**，前一版本全部验收通过后才进入下一版本
2. **界面语言**：仅中文
3. **视觉风格**：深色工业风主题，参考 V0.1 任务书中的 Theme.qml
4. **Git 提交规范**：每完成一个版本做一次提交（中文提交说明），推送到 github 和 gitee 双平台
5. **验收标准**：每个版本完成后必须运行对应验收检查单
6. **参考项目**：在 `reference/` 目录，学习实现方法和交互逻辑，取其精华
7. **开发文档变更**：规划文档的修改也要纳入 Git 提交
8. **扩展性**：所有接口设计必须预留后续功能扩展的冗余空间
